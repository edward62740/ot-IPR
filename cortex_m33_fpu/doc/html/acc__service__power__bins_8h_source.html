92 /**
93  * @brief Set the sensor downsampling factor
94  *
95  * Sets the downsampling factor - the number of steps taken between each data point. A downsampling factor of 1 samples
96  * every possible point in the range. A downsampling factor of 2 samples every other point, and so on.
97  *
98  * The power bins service supports a downsampling factor of 1, 2, or 4.
99  *
100  * In power bins, the downsampling factor does not affect the resulting data length.

104  */
