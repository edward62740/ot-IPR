23 /**
24  * @brief Activate the Acconeer Radar System Software, RSS
25  *
26  * A HAL struct containing integration functions (such as 'wait_for_interrupt', 'mem_alloc' and 'log')
27  * needed by RSS must first be populated and then sent in. See 'acc_definitions_common.h' for a full list

30  * This function must be called before any other RSS function. If it is not, or it failed,
31  * calling any other RSS function is undefined behaviour.

34  * @return True if RSS activated successfully
