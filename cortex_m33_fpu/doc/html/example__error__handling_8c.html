Service Envelope Error Handling example.

Returns
Returns EXIT_SUCCESS if successful, otherwise EXIT_FAILURE

Return value == false is assumed to be due to a sensor_communication_error, which is handled in handle_indication. Assume that the same data length can be used regardless of indication.
