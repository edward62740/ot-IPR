Acconeer SDK Documentation

Acconeer SDK provides software for easy development of applications using the A111 sensor. The SDK consists of

  • Radar System Software, RSS
  • HAL reference implementation
  • Example and reference applications

Radar System Software provides four different services. A service enables the client to configure the sensor to retrieve pre-processed data which can be used for multiple purposes. More information regarding services can be read in the sensor introduction document. The data from each service can also be visualized in the Exploration Tool.

Acconeer also implements detectors which utilize the services and the respective output data. A detector adds further processing of the data for typical use cases such as distance measurement, presence detection and obstacle detection. The detectors are implemented to be used in applications which just need a single response from the radar with for example distance to an object and the user doesn't want to implement the signal processing themselves.

The example applications show how to use the RSS API and the different services and detectors. It could be used as a base for users to start implementing their own application. A reference application is further developed to show how to use a specific service or detector for a typical use case.
