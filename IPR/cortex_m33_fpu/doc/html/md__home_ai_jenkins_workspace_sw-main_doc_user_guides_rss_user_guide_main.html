User guides

User guides include a getting started guide for each of Acconeer's services and detectors. They describe how to get started with an easy example to retrieve data from the sensor which can be used in various applications. It is recommended to use the Exploration Tool for better understanding of the output data from the services and how different configuration parameters affect the data.

Services

Acconeer provides four different services in the Radar System Software SDK. A service enables the client to configure the sensor to retrieve pre-processed data which can be used for multiple purposes. More information regarding services can be read in the sensor introduction document, https://acconeer-python-exploration.readthedocs.io/en/latest/sensor_introduction.html.

Detectors

Acconeer also implements detectors which utilize the services and the respective output data. A detector add further processing of the data for typical use cases such as distance measurement, presence detection and obstacle detection.

Hardware Abstraction Layer

Radar System Software SDK is platform independent and Acconeer deliver libraries for Raspberry Pi, Cortex M7, Cortex M4 and Cortex M0. The application can be implemented on various MCUs and the platform specific functions needed are defined in the Hardware Abstraction Layer, HAL. The User Guide HAL Software Integration describe the HAL and how to integrate in on a specific MCU.
