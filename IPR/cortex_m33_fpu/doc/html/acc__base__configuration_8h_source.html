196 /**
197  * @brief Get the hardware accelerated average samples (HWAAS)
198  *
199  * Each data point can be sampled between 1 and 63 times, inclusive, and the sensor hardware then
200  * produces an average value of those samples. The time needed to measure a sweep is roughly proportional
201  * to the number of averaged samples. Hence, if there is a need to obtain a higher update rate, HWAAS
202  * could be decreased but this leads to lower SNR.
203  *
204  * @param[in] configuration The base configuration to get hardware accelerated average samples from
205  * @return Hardware accelerated average samples
206  */
